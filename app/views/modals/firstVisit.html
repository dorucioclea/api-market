Welcome!

Before you can start using APIs, we need to take care of a few things...

Log in with your A-Profile

The A-Profile provides single log on access to all services in the Antwerp City platform.

Create an Organization

Think of an organization as a container for your applications. It allows you to easily group applications together, and you can invite others to collaborate on those applications.

Start working on a new Application

Applications are what it's really all about. They can have contracts with one or more of our APIs, and an API key will be generated to secure access to those APIs.
